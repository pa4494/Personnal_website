This website all started as a school project and is the result of many hours a Data Science training at Jedha Bootcamp. Over the course of this bootcamp, through a subtil mix of advanced theory and hands on real world projects, we dived deeply into the wide world of Python based data sciences. In particular we were trained on the following technologies and techniques :

  • Data exploration and visualisation
    • Matplotlib
    • Numpy
    • Pandas
    • Seaborn
  • Supervised Machine Learning
    • Linear Regressions
    • Logistic Regressions
    • Cross Validation
    • Ridge and Lasso Regressions
    • Classifier and Regressor Trees
    • Random Forests
    • Naive Bayes
    • SVM
    • Adaboost and XGboost
  • Unsupervised Machine Learning
    • KMeans
    • DBSCAN
  • Deep Learning
    • Gradient descent and loss minimisation theory
    • FastAI
    • Tensorflow 2.0
    • Convolutional Neural Networks
    • NLP and LSTM
    • Transfer Learning
    • Custom training
  • API and web scrapping
    • BeautifulSoup
    • API handling
    • Flask
  • Cloud Computing and Spark
    • Big data and parallel computing
    • Spark SQL
    • Machine Learning with Spark
    • Recommendation engine with Spark
The purpose of this website is to offer simple to use tools to allow users with minimal coding experience and an interest in Data Science to manipulate powerful data visualisation techniques and generate machine learning models in just a few clicks. To do so, I invite you to start by uploading a CSV file of your choice in the Upload section . Once this is done you will have access to the following tools:

  • Data Visualization
    • Univariate Analysis
    • Bivariate Analysis
  • Machine Learning Models
    • Supervised Models
    • Unsupervised Models
  • Applied ML exemples
    • Detect online fraud with feature engineering
    • Website conversion rate optimisation with Trees
    • Assess the impact of a referral campaing
    • Help Uber manage its fleet with Clusterisation
