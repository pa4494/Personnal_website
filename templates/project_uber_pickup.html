# We are first going to try to find clusters with a centroid using the KMeans technic. Given that our dataset
# represent coordinates in a city (NY) we are going to use the manhattan distance. 
# However this distance is not usable with Kmeans in sklearn. Let's first try a simple K-means with Euclidian distance

# First to familiarise with clustering we are going to create clusters based on Latitude and Longitude
sample_data1 = sample_data.drop(["Month","Day", "Hour"], axis =1)

# We start by normalising our variables with a MinMax scaler

from sklearn.preprocessing import MinMaxScaler
MinMaxS = MinMaxScaler()
sample_data1 = pd.DataFrame(MinMaxS.fit_transform(sample_data1), columns = sample_data1.columns)

# Then we use Kmeans. Let's say we want to divide the whole pick up zone in clusters and we'd like to figure out how manies
# To help us decide, let's do a elbow chart analysis
# To process the data faster we are using MiniBatchKmean which is an implementation of K means only slightly less precise
# but a lot faster, especially with large datasets

from sklearn.cluster import MiniBatchKMeans

Sum_of_squared_distances = []
K = range(8,20)
for k in K:
    km = MiniBatchKMeans(n_clusters=k, random_state = 42, )
    km = km.fit(sample_data1)
    Sum_of_squared_distances.append(km.inertia_)  # ici on peut utiliser .score au lieu d'inertia_
     
plt.plot(K, Sum_of_squared_distances, 'bx-')
plt.xlabel('k')
plt.ylabel('Sum_of_squared_distances')
plt.title('Elbow Method For Optimal k')
plt.show()

Conclusion : 12 clusters seems like a decent choice

What this seems to tell us is that pick up are mostly distributed throughout the day rather than throughout long and lat. In other terms, assuming a finite number of drivers, uber would be better off having throughout the whole day rather than distributed all over Manhattan. Of course operationnally speaking this result is not satisfying and we would like to have geographical clusters for each period of the day. Assuming a base shift of 8 hour for a driver, let's divide the day in 3 periods.
